## HJS Project Overview

**HJS** is a **JavaScript dialect with native HTML support**, built on top of [XJS](https://github.com/xjslang/xjs), an experimental parsing tool.

### Main Concept
It allows writing JavaScript code where you can include HTML elements directly in the syntax, similar to JSX but with its own compiler written in Go.

**Example syntax:**
```javascript
let p = <p>
  "Hello, "
  <strong>"World!"</strong>
</p>
```

This compiles to vanilla JavaScript that creates DOM elements:
```javascript
let p = (function(){
  const elem = document.createElement('p');
  elem.append("Hello, ");
  elem.append((function(){
    const elem = document.createElement('strong');
    elem.append("World!");
    return elem;
  })());
  return elem
})();
```

### Project Architecture

The project is organized in Go and has these main components:

1. **hjs.go** - Entry point with 3 main functions:
   - `Parse()` - Transforms HJS code to AST
   - `Compile()` - Transforms AST to valid JavaScript code
   - `Format()` - Transforms AST to formatted code

2. **compiler.go** - Compiler that transforms HTML tags to `document.createElement()` calls
   - Handles regular attributes with `setAttribute()`
   - Handles events (`onClick`, etc.) by converting them to `addEventListener()`

3. **formatter.go** - Code formatter

4. **plugin.go** - Plugin that extends the XJS parser

5. **Web Playground** - Available at https://xjslang.github.io/hjs/
